# gatsby-plugin-outbrain

Gatsby plugin for adding Outbrain tracking code

## Install

`npm install --save gatsby-plugin-outbrain`

## How to use

```javascript
// In your gatsby-config.js
plugins: [
  {
    resolve: `gatsby-plugin-outbrain`,
    options: {
      outbrainId: "__ID__HERE__",
    },
  },
];
```
